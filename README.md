# NGO Web Application Demo (CSE 310)

## Overview
This project is a simple **Web Application** created for the **CSE 310 â€“ Applied Programming** course.  
The goal of this project is to demonstrate how a basic web application works using **HTML, CSS, and JavaScript**.

The website represents a demo NGO platform where users can navigate through different pages and interact with forms such as contact and donation pages.

This project focuses on front-end development and user interaction.

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

No backend or database is used. All interactions are handled on the client side.

## Features
- Multi-page website (Home, About, Candidates, Manifesto, Contact, Donate)
- Responsive layout using CSS
- JavaScript for:
  - User interaction
  - Form handling
  - Basic validation
  - Page behavior and animations
- Clean and commented code for readability

## How to Run the Project
1. Download or clone this repository to your computer.
2. Open the project folder in **Visual Studio Code**.
3. Open `index.html` in your browser  
   - Or use the **Live Server** extension in VS Code for a better experience.
4. Navigate through the site using the menu links.
